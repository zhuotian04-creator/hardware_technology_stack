# Networking and IoT

One of ESP32's biggest strengths is wireless connectivity. When learners reach this stage, the course moves from local control to connected devices.

## 1. What This Stage Covers

### 1.1. Order

1. [Wi-Fi](./wifi.md)
2. [Bluetooth](./bluetooth.md)
3. [Web Server](./web-server.md)
4. [MQTT](./mqtt.md)

### 1.2. Learning Goals

After this stage, learners should be able to:

1. Connect ESP32 to Wi-Fi
2. Turn ESP32 into an access point
3. Control the board from a web page
4. Publish and subscribe with MQTT
5. Understand the basics of Bluetooth use cases

## 2. Key Focus

### 2.1. Wi-Fi

Used for networking, online data, and cloud platforms.

### 2.2. Web Control

Lets phones and computers control the board directly in a browser.

### 2.3. MQTT

One of the most common IoT message protocols for telemetry and remote control.
