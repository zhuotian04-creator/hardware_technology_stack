# 网络与物联网

ESP32 的优势之一就是无线连接能力。进入这一阶段后，课程会把设备从“单机控制”带到“联网控制”。

## 1. 本阶段内容

### 1.1. 学习顺序

1. [Wi-Fi](./wifi.md)
2. [蓝牙](./bluetooth.md)
3. [Web 服务器](./web-server.md)
4. [MQTT](./mqtt.md)

### 1.2. 学习目标

完成这一阶段后，学习者应该能够：

1. 连接 Wi-Fi 网络
2. 把 ESP32 变成热点
3. 用网页控制开发板
4. 用 MQTT 上报和订阅数据
5. 了解蓝牙的基础使用场景

## 2. 课程重点

### 2.1. Wi-Fi

用于联网、获取在线数据和对接云平台。

### 2.2. Web 控制

让手机或电脑通过浏览器直接控制设备。

### 2.3. MQTT

最常见的物联网消息协议之一，适合设备上报和远程控制。
