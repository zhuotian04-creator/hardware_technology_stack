# 执行器

执行器负责把 ESP32 的控制信号变成现实世界中的动作。

## 1. 常见执行器

### 1.1. 继电器

适合控制风扇、灯、泵等较大功率负载。

### 1.2. 舵机

适合做角度控制、机械臂、指向机构。

### 1.3. 直流电机

适合风扇、小车、传送结构等需要连续旋转的场景。

### 1.4. 步进电机

适合需要较精确位置控制的场景。

## 2. 控制方式

| 执行器 | 常见控制方式 | 通常需要的外部器件 |
| --- | --- | --- |
| LED、蜂鸣器 | 数字输出或 PWM | 限流电阻、三极管视功率而定 |
| 继电器 | 数字输出 | 带驱动和续流保护的继电器模块 |
| 舵机 | 50Hz 控制脉冲 | 独立 5V 电源 |
| 直流电机 | PWM 和方向信号 | TB6612、DRV8833 等电机驱动 |
| 步进电机 | 步进和方向信号 | A4988、DRV8825 等驱动器 |

GPIO 只负责发出控制信号，不负责给大功率负载供电。电机、舵机、继电器线圈等设备必须通过合适的驱动和电源工作。

## 3. 舵机最小示例

在 Arduino IDE 的库管理器中安装 `ESP32Servo`。示例使用 `GPIO 18`，让舵机在三个角度之间移动。

```cpp
#include <ESP32Servo.h>

Servo servo;
const int servoPin = 18;

void setup() {
  servo.setPeriodHertz(50);
  servo.attach(servoPin, 500, 2400);
}

void loop() {
  servo.write(0);
  delay(1000);

  servo.write(90);
  delay(1000);

  servo.write(180);
  delay(1000);
}
```

小舵机也可能产生较大的瞬时电流。应使用独立 5V 电源给舵机供电，并把外部电源 GND 与 ESP32 GND 相连。不要用开发板的 3.3V 引脚给舵机供电。

## 4. 推荐实验顺序

1. 先用 LED 验证数字输出和 PWM。
2. 再用蜂鸣器验证频率或开关控制。
3. 使用独立电源完成舵机角度控制。
4. 通过驱动模块控制直流电机方向和速度。
5. 最后加入传感器，完成自动联动。

## 5. 重要提示

1. 控制信号和负载电源不是一回事。
2. 驱动芯片通常是必要的。
3. 不要让 GPIO 直接去带大电流设备。
4. 感性负载需要续流保护，优先使用成熟驱动模块。
5. 上电前检查电源极性、额定电压和共地。
6. 电机造成重启时，先排查供电压降和干扰，不要只改程序。

## 6. 联动时的程序要求

1. 给动作设置最大持续时间，避免设备失控后一直运行。
2. 为频繁开关增加最小间隔。
3. 把“目标状态”和“实际输出”分别记录到串口。
4. 网络控制失联后应进入预先定义的安全状态。
