# ESP32 基础认知

这一页帮助学习者先认识 ESP32 开发板，而不是一上来就直接写项目。

只要把开发板、电源、电平、引脚和一些常见限制搞清楚，后面的 GPIO、传感器和通信协议会更容易理解。

## 1. ESP32 是什么

ESP32 是 Espressif 推出的一系列低成本、低功耗微控制器，常见特点包括：

- 内置 Wi-Fi
- 内置蓝牙
- 丰富的 GPIO
- 支持 ADC、PWM、UART、I2C、SPI 等外设

它很适合做：

- 智能硬件
- 物联网设备
- 传感器采集
- 无线控制
- 小型嵌入式系统

## 2. 先认识这些概念

### 2.1. GPIO

GPIO 是最基础的通用输入输出引脚，用来接按钮、LED、继电器、蜂鸣器等设备。

### 2.2. 电平

ESP32 的 GPIO 工作电压通常是 `3.3V`。

这意味着：

1. 大多数引脚不适合直接接 `5V` 信号
2. 连接 `5V` 模块时要特别注意电平转换
3. 不能把 `5V` 直接加到 `3.3V` 引脚上

### 2.3. 输入专用引脚

有些引脚只能作为输入使用，常见于高位 ADC 或特殊用途场景。

### 2.4. 启动相关引脚

有些引脚会影响芯片启动模式，接外设时要避免把它们拉到错误电平。

## 3. 常见外设能力

| 能力 | 用途 |
| --- | --- |
| GPIO | 控制输入输出 |
| ADC | 读取模拟信号 |
| PWM | 控制亮度和转速 |
| UART | 串口通信 |
| I2C | 连接传感器和扩展模块 |
| SPI | 连接高速外设 |
| Wi-Fi | 无线联网 |
| Bluetooth | 设备近距离通信 |

## 4. 开发板供电

### 4.1. 常见供电方式

ESP32 开发板一般会通过以下方式供电：

- USB 供电
- 5V 引脚输入
- 3.3V 引脚输入

### 4.2. 注意事项

1. 先确认开发板是否已经带稳压电路
2. 不要把 `5V` 直接接到 `3.3V` 引脚
3. 不要把外设电流需求算得太乐观
4. 如果一个项目要接很多模块，先做电源预算

## 5. 学完这一页后要记住什么

完成这一页后，学习者至少应该知道：

1. ESP32 是一块什么样的开发板
2. `3.3V` 电平意味着什么
3. GPIO 不只是“插线就能用”
4. 为什么有些引脚不能随便接
5. 开发板供电不是越随意越好
