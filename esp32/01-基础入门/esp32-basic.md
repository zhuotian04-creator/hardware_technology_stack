# ESP32 基础知识

ESP32 是由乐鑫信息科技（Espressif Systems）推出的一款低成本、低功耗的双核微控制器，集成了 Wi-Fi 和双模蓝牙功能。

## 1. 核心规格参数

| 参数 | 规格描述 |
| :--- | :--- |
| **处理器** | Tensilica Xtensa Dual-Core 32-bit LX6 处理器，主频高达 240 MHz |
| **SRAM** | 520 KB |
| **外部 Flash** | 通常为 4 MB 或 8 MB（取决于具体模组） |
| **无线连接** | Wi-Fi 802.11 b/g/n, 蓝牙 v4.2 BR/EDR 和 BLE (低功耗蓝牙) |
| **常用外设接口** | GPIO, ADC, DAC, PWM, UART, SPI, I2C, I2S, Touch Sensor |

## 2. 常见引脚功能 (Pinout)

ESP32 芯片拥有丰富的 GPIO 引脚，但并非所有引脚都适合用作普通输入输出：

* **仅限输入引脚 (Input Only GPIOs)**: GPIO 34, 35, 36, 39。这些引脚没有内部上拉/下拉电阻，且只能作为输入使用，常用于 ADC 模拟量采集。
* **straps 引脚 (Strapping Pins)**: GPIO 0, 2, 5, 12, 15。这些引脚在芯片复位或上电时用于确定启动模式，开发时应避免在外围电路中强行拉高或拉低它们，否则可能会导致无法烧录或无法正常启动。
* **内置 LED 引脚**: 大多数 ESP32 开发板（如 ESP32 NodeMCU）在 GPIO 2 上连接了一个蓝色板载 LED。
* **ADC 引脚**:
  * **ADC1**: GPIO 32-39（推荐使用，不受 Wi-Fi 使用影响）。
  * **ADC2**: GPIO 0, 2, 4, 12-15, 25-27（当启用 Wi-Fi 时，ADC2 将不可用）。

## 3. 工作电压与供电注意事项

> [!WARNING]
>
> * ESP32 芯片的 GPIO 引脚工作电压为 **3.3 V**。输入超过 3.6 V 的电压可能会损坏引脚或整个芯片。
> * 在连接 5 V 的传感器或器件时，必须进行电平转换（例如使用电平转换模块或电阻分压）。
> * 开发板通常提供 5V（来自 USB）或 3.3V（经板载 LDO 稳压芯片降压后）的输出引脚，请谨慎接线，严禁将 5V 直接接到 3.3V 引脚。
