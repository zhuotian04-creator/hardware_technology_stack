# 综合项目

这一阶段用来把前面学到的内容整合成完整项目。训练营走到这里，重点不再是“学了几个知识点”，而是“能不能独立做出一个系统”。

## 1. 本阶段内容

### 1.1. 学习顺序

1. [项目需求](./project-requirements.md)
2. [项目检查清单](./project-checklist.md)

### 1.2. 学习目标

完成这一阶段后，学习者应该能够：

1. 读懂项目需求
2. 拆分项目模块
3. 给项目做功能检查
4. 完成一个基础的综合硬件作品

## 2. 适合做什么项目

- 智能灯光
- 环境监测
- 小型物联网终端
- Web 控制装置
- 传感器联动小系统
